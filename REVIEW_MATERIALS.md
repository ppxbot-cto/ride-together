## 🎯 小程序介绍文案

### 简短介绍（120 字以内）
```
骑行记 RT 是一个骑行活动分享平台，用户可以发布骑行活动信息，浏览和加入感兴趣的活动。支持摩托车和自行车两种车型，让车友找到趣味相投的伙伴一起出发。
```

### 详细介绍（用于页面展示）
```
骑行记 RT 是专为摩托车和自行车爱好者打造的骑行活动社交平台。

【核心功能】
- 发布骑行活动：分享你的骑行计划，包括时间、地点、路线
- 浏览活动：查看附近的车友发布的活动
- 车型筛选：支持摩托车和自行车分类
- 一键加入：感兴趣的活动点击即可报名
- 个人中心：管理我发布的和我加入的活动

【使用场景】
- 周末想骑车找不到伴？来骑行记发布活动！
- 刚到陌生城市想骑行？浏览当地活动加入！
- 发现有趣骑行路线？分享给车友一起探索！

骑行记，不孤单！🏍️🚴
```

```markdown
5.1 平台尊重用户隐私，具体隐私政策见《隐私政策》文档。

6.1 因不可抗力导致的服务中断，平台不承担责任。

7.1 平台有权根据需要修改本协议，修改后会公示。
7.2 用户继续使用服务视为接受修改后的协议。
```

## 📄 隐私政策

```markdown
# 骑行记 RT 隐私政策

## 7. 政策更新

7.2 重大变更会通过小程序公告通知。
```
